# ISBUSHKA — Salon Management System (Case Study)

ISBUSHKA is a pet-project and case study that demonstrates full-cycle
business and system analysis for a small service business.

The project is focused on structured documentation rather than code
and represents how a Business/System Analyst approaches a real-world product
from idea to scalable system.

---

## Project Goals

- Demonstrate business analysis and system analysis skills
- Showcase structured documentation artifacts
- Model business processes, data, and business rules
- Provide a reusable reference for future projects

---

## Project Scope

**In scope**
- Client management
- Appointment scheduling
- Financial operations tracking
- Reporting and analytics

**Out of scope**
- Marketing automation
- External payment processing (conceptual only)
- Mobile native development

---

## Artifacts Included

- Business process models (BPMN)
- Use cases (UC) and user stories (US)
- Functional and non-functional requirements (FRS, NFRS)
- Data model (ERD) and data dictionary
- Business rules and calculations
- Roadmap and change log

---

## Tools & Technologies

- AppSheet (no-code platform)
- Google Sheets (data storage)
- BPMN / ERD
- Markdown documentation
